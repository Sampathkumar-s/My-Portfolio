About the Project

Smart AgroTech is an integrated IoT and AI-based solution for modernizing agriculture. It uses field sensors to monitor soil moisture, temperature, and weather, providing real-time updates to farmers.

Crop-specific advisory is generated using AI models based on sensor data and external API feeds. Designed for mobile-first interfaces, it supports offline syncing and regional language guidance to enhance farmer accessibility.
